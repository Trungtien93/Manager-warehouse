# Notification Sound Files

Place the following audio files in this directory:

- `notification-default.mp3` - Default notification sound
- `notification-urgent.mp3` - Urgent/high priority notification sound  
- `notification-info.mp3` - Low priority/info notification sound

## Notes

- Supported formats: MP3, WAV, OGG
- Recommended duration: 0.5-2 seconds
- Recommended volume: Moderate (not too loud)

You can find free notification sounds at:
- https://notificationsounds.com/
- https://mixkit.co/free-sound-effects/notification/
